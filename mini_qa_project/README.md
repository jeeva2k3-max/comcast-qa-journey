\# Mini QA Project – User Eligibility Validator



\## Purpose

This project simulates a QA automation scenario where user eligibility

is validated using external test data.



\## What it covers

\- File-based test data

\- Data parsing

\- Edge case handling

\- Defensive programming

\- Business rule validation



\## Files

\- users\_data.txt – test input data

\- user\_validator.py – validation logic



\## Output

The script prints eligibility results for each user record without crashing,

even when data is invalid.
